# Welcome to pyospackage_jingyuan

pyospackage_jingyuan is a project that adds or multiplies numbers together.

## Get started

You can install this package into your preferred Python environment using pip:

```bash
$ pip install pyospackage_jingyuan
```

or directly from GitHub using:

```bash
$ python -m pip install git+https://github.com/ljy0401/pyospackage_jingyuan
```

To use pyospackage_jingyuan in your code:

```python
>>>  from pyospackage_jingyuan.calculate import add_num, multiply_num
>>>  add_num(3,-99)
>>>  multiply_num(33,-2)
```
